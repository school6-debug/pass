# Random Password Generator

## Автор
СОШ №6 Нефтекамск

## Описание
Это приложение позволяет генерировать случайные пароли с настройками длины и выбора символов (строчные и заглавные буквы, цифры, спецсимволы). Также сохраняется история сгенерированных паролей.

## Примеры использования
1. Установите длину пароля с помощью ползунка.
2. Выберите необходимые параметры (заглавные буквы, цифры, спецсимволы).
3. Нажмите кнопку «Сгенерировать пароль» для получения нового пароля.
4. Сгенерированные пароли будут отображаться в списке истории.

## Установка
1. Клонируйте репозиторий.
2. Убедитесь, что у вас установлен Python.
3. Запустите приложение с помощью команды python pass.py.
